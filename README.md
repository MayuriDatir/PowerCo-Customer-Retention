# PowerCo Customer Retention Analysis

## Problem
Customer churn analysis for PowerCo.

## Dataset
Telco Customer Churn Dataset (IBM) from Kaggle
[Link](https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset)

## Approach
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Churn Prediction Models (Logistic Regression, Random Forest)
5. Business Insights & Retention Strategy

## Key Insights
- Month-to-month contracts have highest churn
- Customers with low tenure and high bills are most likely to churn
- Targeted retention campaigns recommended

## Tools
Python, Pandas, Scikit-learn, Seaborn, Matplotlib
